# Geogotchi

Library for working with [GeoNames][geonames] services.

## Usage

```python
>>> from pprint import pprint
>>> from geogotchi import Geogotchi
>>> 
>>> lkpg = (58.411, 15.622)
>>> gg = Geogotchi(username="demo")
```
